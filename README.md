					API para seleção de imóveis


Como utilizar:
	Execute o servidor:
		python3 server.py

	Acesse as urls:
		Zap:
		http://127.0.0.1:8001/api/?portal=zap

		VivaReal:
		http://127.0.0.1:8001/api/?portal=zap
		
	Para evitar que a porta 8001 fique bloqueada, feche o servidor com CTRL + C
