					A simple python server for selecting json data

How to use:
	Execute the server:
		python3 server.py

	Access the urls:
		Zap:
		http://127.0.0.1:8001/api/?portal=zap

		VivaReal:
		http://127.0.0.1:8001/api/?portal=zap
		
	For avoidind the door 8001 staying blocked close the server with CTRL + C
